## 사전 렌더링 

pre Randering

브라우저의 요청에 사전에 렌더링이완료된 HTML을 응답하는 렌더링 방식

1. 초기 접속 요청을 보냄
2. 서버가 js 실행 자바스크립트 코드를 HTML로 변환
3. 렌더링이 된 html을 브라우저로 보냄
4. 브라우저는 유저에게 화면을 보여줌

FCP 가 굉장히 빠름

5. js bundle 보냄 브라우저는 HTML에 JS를 연결 (HYDRATION) 

6. 상호작용가능
